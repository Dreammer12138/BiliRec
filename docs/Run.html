运行

运行

Windows

PS > python start.py -r <roomid> -u <uid> -p <port> -o <outpath>

Linux

$ sudo python3 -r <roomid> -u <uid> -p <port> -o <outpath>

参数

-r / --room 房间号

-u / --uid 主播UID

-p / --port 端口

-o / --outpath 输出路径

Web端监控

启动脚本以后打开浏览器，访问http://localhost:<port>
